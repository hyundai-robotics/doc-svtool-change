# 2.5 접속/분리 명령(toolchng)






- 설명 
    
    ```toolchng```는 작업 프로그램 실행에 의한 서보툴 체인지 명령어입니다.

- 문법
  
    - ```toolchng``` on/off,tg=<체인지 대상>,di=<접속완료 신호>,wait=<접속완료 대기시간>, mc=<체인지 대상>

- 파라미터
  
   ① on/off
     - on : 서보툴 접속
     - off :  서보툴 분리
   
   ② 체인지 대상
     - G1~G16 : 접속/분리할 용접건 번호
     - P1~P16 : 접속/분리할 포지셔너 번호
     - J1~J16 : 접속/분리할 지그 번호

   ③ 기계적 접속완료 확인신호

     - 1~4096 : 기계적인 접속완료 확인을 위한 입력신호 번호

     - 내용(vol_offset) : Arc용접 시 시너직 전압의 옵셋 전압 값
     - 범위(vol) : 20 ~ 40 V
     - 범위(vol_offset) : 200 ~ 200 V(%)

   ④ 접속완료 대기시간

     - <0~5.0> (sec) : 접속완료 대기시간 (파라미터가 없거나 0이면 무한대기)

  
   ⑤ 체인지 대상 (동시 접속/분리)

     - G1~G16 : 접속할 용접건 번호
     - P1~P16 : 접속할 포지셔너 번호
     - J1~J16 : 접속할 지그 번호
